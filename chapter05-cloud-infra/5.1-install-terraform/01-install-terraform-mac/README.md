## Install Terraform

1. Open Terminal

2. Tap hashicorp/tap
```bash
brew tap hashicorp/tap
```

3. Brew Install
```bash
brew install hashicorp/tap/terraform
```

4. Verify Terraform
```bash
terraform --version
```
